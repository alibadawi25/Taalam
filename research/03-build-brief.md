# 03 Build Brief — Yanfaa Website (Approval Draft)

## Context Summary
Yanfaa already communicates a strong affordability proposition (“all courses with one subscription”), but top competitors outperform on **outcome framing**, **early trust proof**, and **guided pathway UX**.

This brief converts those findings into a concrete execution plan.

---

## Design Direction

### Recommended Color Palette (brand-refined)
- Primary: `#1ab394`
- Primary dark: `#0f8f78`
- Secondary deep: `#252541`
- Accent: `#09d491`
- Surface light: `#f7faf9`
- Text main: `#121826`

### Typography Pairing
- Headings: **IBM Plex Sans Arabic** (already present in Yanfaa stack) with stronger weight hierarchy.
- Body: **IBM Plex Sans Arabic** (400/500) for consistency and performance.

### Photography / Visual Asset Style
- Real learner/workspace imagery over abstract stock.
- Outcome visuals by track (design, marketing, programming) with subtle category color coding.

### Animation Guidance
- Scroll-triggered section reveals (mild translate + opacity).
- Sticky pathway selector in homepage mid-section.
- Soft hover elevation on cards/buttons.
- Avoid heavy parallax that harms performance on mobile.

### What to Avoid
- Overcrowded hero with too many equal CTAs.
- Long ungrouped course lists without intent filters.
- Dark-heavy UI that reduces readability for Arabic body copy.

---

## Site Architecture

## Primary Pages
1. **Home** — value prop + pathway discovery + proof + CTA.
2. **Learning Paths** — role-based journeys (SEO, UI/UX, Media Buying, etc.).
3. **Course Catalog** — filterable exploration for direct course discovery.
4. **Pricing / Subscription** — one clear plan + FAQ + conversion reassurance.
5. **Support / FAQ** — trust, help, payment and access answers.

### Navigation Structure
- Home
- Learning Paths
- Browse Courses
- Pricing
- Login / Start Now

### Content Hierarchy (Home)
1. Hero: one outcome-led headline + one primary CTA.
2. Trust strip: learners count / instructor credibility / completion signal.
3. “Choose your path” role cards.
4. Featured courses from each major category.
5. Testimonials / success stories.
6. Final CTA with subscription clarity.

### CTA Strategy
- Primary CTA: **ابدأ الآن**
- Secondary CTA: **استكشف المسارات**
- Repeated primary CTA in hero, after pathways, and final block.

## Content Framework

### Homepage Headline Options (proven formula style)
1. **اتعلّم مهارة عملية وابدأ تطبيقها خلال أسابيع — باشتراك واحد فقط.**
2. **من أول خطوة للاحتراف: مسارات تعلم واضحة في منصة عربية واحدة.**
3. **بدل تشتت الكورسات: كل ما تحتاجه للتطور المهني في اشتراك واحد.**

### Value Proposition Structure
- Problem: كثرة مصادر التعلم وتشتت المحتوى.
- Promise: مسارات واضحة منظمّة حسب الهدف الوظيفي.
- Mechanism: مكتبة كورسات + مسارات + متابعة تقدم.
- Offer: اشتراك واحد مناسب الميزانية.

### Section-by-Section Copy Direction
- Hero: outcome + affordability.
- Path section: “What do you want to become?”
- Proof section: numbers + testimonials.
- Pricing: simplicity + no hidden complexity.

### SEO Targets
- المنصة التعليمية العربية
- دورات اونلاين
- تعلم عن بعد
- مسارات تعلم
- دورات [تخصص] للمبتدئين
- تعلم [مهارة] من الصفر

## Conversion Playbook

### Primary Conversion Goal
- Subscription start (trial/paid start flow depending on product policy).

### Lead Capture Strategy
- Soft capture in path flow (“احفظ مسارك وابدأ لاحقًا”).
- Exit intent modal only on desktop (lightweight, non-intrusive).

### Social Proof Plan
- Near-hero stats (learners, course hours, paths completed).
- Mid-page testimonials from target roles.
- Instructor credibility blocks.

### Trust Signal Checklist
- Visible payment/security reassurance.
- Transparent pricing logic.
- Support/FAQ link next to conversion points.
- Terms/privacy accessible from all pages.
